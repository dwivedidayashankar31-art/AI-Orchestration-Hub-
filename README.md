  ## 🌟 Overview

  **CloudVault** is a complete Enterprise SaaS frontend platform designed for modern businesses. It comes with a premium dark-themed UI, smooth animations, and 15+ pre-built pages — ready to plug into any backend.

  Whether you're building a startup SaaS product, an admin panel, or a company dashboard, CloudVault gives you a professional head start.

  ---

  ## 🚀 Features

  - **Premium Dark Theme UI** — Sleek, modern design built for SaaS applications
  - **Fully Responsive** — Pixel-perfect on Desktop, Tablet, and Mobile
  - **15+ Ready-to-Use Pages** — Everything you need, already built
  - **Smooth Animations** — Powered by Framer Motion for a polished feel
  - **Clean Codebase** — Well-organized and easy to customize
  - **Component Library** — High-quality custom UI components (Shadcn UI inspired)
  - **Type-Safe** — Built with TypeScript for reliability and developer confidence
  - **Fast Performance** — Vite-powered build for lightning-fast development and production

  ## 📦 Getting Started

  ### Prerequisites

  Make sure you have [Node.js](https://nodejs.org/) (v18+) installed.

  ### Installation

  ```bash
  # 1. Clone the repository
  git clone https://github.com/dwivedidayashankar31-art/CloudVault---Enterprise-SaaS-Platform.git

  # 2. Navigate to the project directory
  cd CloudVault---Enterprise-SaaS-Platform

  # 3. Install dependencies
  npm install

  # 4. Start the development server
  npm run dev
  ```

  Open your browser and visit `http://localhost:5000`

  ### Build for Production

  ```bash
  npm run build
  ```

  ## 🎨 Customization

  CloudVault is built to be easily customizable:

  - **Colors & Theme** — Edit CSS variables in `src/index.css`
  - **Fonts** — Update font imports in the global CSS
  - **Tailwind Config** — Extend or override in `tailwind.config.ts`
  - **Pages** — Add or remove pages from `src/pages/` and update routing in `App.tsx`
  - **Components** — Reuse or extend components from `src/components/`
