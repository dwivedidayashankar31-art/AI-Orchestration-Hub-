# CloudVault - Enterprise SaaS Platform

CloudVault is a modern, premium, and fully responsive Enterprise SaaS frontend template. Built with the latest web technologies, it provides a clean, dark-themed UI perfect for dashboards, admin panels, and SaaS landing pages.

## 🚀 Features

- **Premium Dark Theme UI:** Designed for modern SaaS applications.
- **Fully Responsive:** Looks perfect on Desktop, Tablet, and Mobile.
- **15+ Ready-to-Use Pages:**
  - Landing Page
  - User Dashboard
  - Admin Panel
  - Company Dashboard
  - Pricing
  - Features
  - Documentation (Docs)
  - Blog
  - Contact & About Us
  - Authentication (Login/Signup)
  - Checkout
- **Smooth Animations:** Integrated with Framer Motion for a polished feel.
- **Clean Codebase:** Easy to customize and connect to any backend.
- **Component Library:** Built using high-quality custom UI components (Shadcn UI inspired).

## 🛠️ Tech Stack

- **Framework:** React.js
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Routing:** Wouter
- **State Management:** React Query (Ready for backend integration)

## 📦 Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your system.

### Installation

1. **Clone or Download** this repository.
2. **Navigate** to the project directory in your terminal.
3. **Install dependencies:**
   ```bash
   npm install
   ```

### Running the Development Server

Start the local development server:

```bash
npm run dev
```

The application will start, and you can view it in your browser at `http://localhost:5000`.

## 🎨 Customization

This template is designed to be easily customizable. You can change colors, fonts, and styles by modifying the Tailwind configuration (`tailwind.config.ts`) and global CSS variables (`client/src/index.css`).
